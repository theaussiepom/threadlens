# Code of Conduct

ThreadLens follows a simple standard: be respectful and constructive.

We expect participants to:

- Use welcoming and inclusive language
- Accept constructive criticism gracefully
- Focus on what is best for the project and its users
- Show empathy toward other contributors

Unacceptable behaviour includes harassment, trolling, personal attacks, and publishing others' private information without permission.

Maintainers may remove, edit, or reject contributions that violate this code and may ban repeat offenders.

Report conduct concerns via a GitHub issue (without secrets) or the security contact path in [SECURITY.md](SECURITY.md).
